# Home Landing Page Design

## Description

Design and implement a standalone home landing page for BIM760 insurance project. The page will incorporate Payload CMS building blocks (navbar, footer) while being a static page. Features include a hero section with slogan and 8 cards, static content sections, insurance flow steps, FAQ accordion, and footer.

## Steps

1. Review existing Payload blocks and components structure
2. Research Payload CMS block creation patterns using Context7
3. Create new hero block component with full-screen background, slogan, and 8 cards
4. Implement static content sections (content + image, image + content)
5. Build insurance flow section with 4 numbered steps
6. Create FAQ accordion component with 5 questions
7. Assemble the home page layout in page.tsx using navbar, hero block, content sections, flow, FAQ, and footer
8. Ensure responsive design and SEO optimization
